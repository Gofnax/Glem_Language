**Data Types:**<br>
In Glem we support the usage of integers and boolean values.<br/>
With this, you can use basic arithmetic operations:
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (integer division) (/)
* Modulo (%)

and boolean operations:
* AND (&&)
* OR (||)
* NOT (!)
